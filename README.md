# Translator

This is a nearly real time real time language translator.

Libraries required:
1)speech_recognition : pip install SpeechRecognition
2)googletrans : pip install googletrans
3)gtts : pip install gtts

How it works?
>The start by prompting to say 'hello'. 
>Once the user says hello the further program is triggered or else the program is terminated.
>After the program is triggered it then prompts the user to speak the sentence he/she wants to translate.
>The program recognizes the sentence converts it into text and displays the user to verify it.
>The text is then translated in the desired language.
>Finally the translated text is converted to an audio file and is saved in the same folder.
